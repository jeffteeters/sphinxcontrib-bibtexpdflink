Installation
============

Install the extension with:

.. code-block:: shell

    $ pip install sphinxcontrib-bibtexpdf

Add the extension to the ``conf.py`` file to use it:

.. code-block:: shell

    extensions = [
        "sphinxcontrib.bibtex",
        "sphinxcontrib.bibtex2",
        "sphinxcontrib.bibtexpdf",
        ]
