Usage
=====


Use the normal bibtex or bibtex2 directives:

.. code-block:: rst

   See :cite:`JaeckelLA-1989a` also :footcite:`JaeckelLA-1989a` for more details.

   Some random text...

   .. footbibliography::



This will be rendered as:

See :cite:`JaeckelLA-1989a` also :footcite:`JaeckelLA-1989a` for more details.

Some random text...

.. footbibliography::
